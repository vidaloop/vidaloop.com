Full Stack Software Engineer
San Diego, CA      |      Remote      |      07 / 10 / 2020
About the Role
We are looking for a highly skilled software engineer who is comfortable with both front and back end programming to help build a secure blockchain-based election system.
Why work at Vidaloop?
  • Make an impact
  • Grow with an early-stage startup
  • Thrive in a flexible, autonomous environment
  • Work with a modern tech stack
What You'll Do
  • Work with the team to build software that matters
  • Design, develop and test our software products and systems
  • Focus on creating end-user value
Skills & Qualifications
Technical Requirements
  • BS in Computer Science or equivalent work experience
  • 4+ years in JavaScript and/or Full Stack development
  • Experience in Modern JavaScript frameworks like Angular and React
  • Understanding of API development in Node.js or other popular frameworks
  • Understanding of schema design in NoSql databases
Strong Pluses
  • Experience developing mobile apps
  • Experience with Amazon Web Services (Cognito, DynamoDB, API Gateway, Lambda, and/or ECS)
  • Understanding of serverless architecture
  • Experience with blockchain
  • Election-industry experience
Other Pluses
  • Thrive in early-stage startup environment
  • Demonstrated leadership ability and personal accountability
  • Self-motivated and resourceful individual
  • Curious, life long learner who enjoys solving interesting problems
  • Bonus: Politically and socially engaged; excited about the elections industry
