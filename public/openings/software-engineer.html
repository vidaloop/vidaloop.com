Software Engineer
Remote (Company based in San Diego, CA)      |      Full Time, Exempt    |      9 / 07 / 2021
About the Role
Vidaloop is looking for a Software Engineer to contribute to our Voterly product. As a member of the Voterly team (https://votingapp.com), your goal will be to develop high-quality software that is aligned with user needs and business objectives.  To be effective in this role, you’ll need experience building modern front end features with a keen sense for usability and accessibility. This role will begin by identifying and collecting usage analytics, as well as expanding our collection of embeddable widgets given to partners.
Why work at Vidaloop?
  • Make an impact in the civic technology space, working to increase informed voter turnout in the United States
  • Get in on the ground floor of an early stage startup
  • Thrive in a flexible, autonomous environment
  • Work with modern technologies
What You'll Do
  • Develop high-quality software design and architecture
  • Identify, prioritize and execute tasks in the software development lifecycle
  • Review and debug code
  • Perform validation and verification testing
  • Build automated tests for new features
  • Ensure software is up-to-date with latest technologies
  • Collaborate with internal teams to fix and improve product
Technology You’ll Use
  • TypeScript/Javascript
  • Node.js
  • Angular
  • Serverless framework
  • AWS (Lambda, DynamoDB, SQS, SNS, API Gateway, Cloudfront)
Strong Pluses
  • Experience identifying and collecting internal analytics and usage data
  • A passion for data organization
  • Experience building web applications that meet accessibility needs
